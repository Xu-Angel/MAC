### 「space」键：快速预览

选中文件后， 不需要启动任何应用程序，使用「space」空格键可进行快速预览，再次按下「space」空格键取消预览。

可以预览mp3、视频、pdf等文件。

我们还可以**选中多张图片**， 然后按「space」键，就可以同时对比预览多张图片。这一点，很赞。

### 翻页和光标

- 「control + ↑」：将光标定位到文章的最开头（翻页到文档的最上方）

- 「control + ↓」：将光标定位到文章的最末尾（翻页到文档的最下方）

- 「control + ←」：将光标定位到当前行的最左侧

- 「control + →」：将光标定位到当前行的最右侧

### 「command + shift + Y」：将文字快速保存到便笺

选中你想要的内容（例如文字、链接等），然后按下「command + shift + Y」，那么你选中的内容就会快速保存到系统自带的「便笺」软件中。

如果你想临时性的保存一段内容，这个操作很实用。

- **调整程序坞呼出时长​**
有不少小伙伴为了获得更大的屏幕空间来放置窗口，选择将程序坞设置为自动隐藏，在这种状态下，将鼠标移动到屏幕底部，稍等一下，程序坞就会出现，但你是否觉得这个程序坞出现前需要等待的时间略微有点长了呢？我来教你如何让程序坞更快的出现。还是请出我们的老朋友，终端，无论你是在启动台搜索终端打开它，还是直接用聚焦搜索终端打开它都可以，接下来输入这行代码，别看到代码就被吓跑，先往后看​
defaults write com.apple.dock "autohide-delay" -float "0" && killall Dock​
代码中被双引号包裹的 0 就是程序坞出现前的等待时间，单位是秒，输入完成后按下回车键确认，墙纸会消失然后恢复，此时你再将鼠标移动到屏幕底部，程序坞立刻就会出现。别小看这么一点时间上的差异，它让你既能拥有更大的桌面空间，又能立刻呼出程序坞，实际使用中会让你感到丝般顺滑。​
如果想要恢复之前的显示速度，只需要输入这行代码​
defaults delete com.apple.dock "autohide-delay" && killall Dock​
按回车运行即可。​

- **无格式粘贴**
当你从网页上复制一段文字，想要粘贴进你的文档中时，这些乱七八糟的格式、字体、字号设置，也会随着文字内容-并被粘贴下来，你还要费时费力地进行调整，因此有不少小伙伴在后台问我，是否可以进行无格式粘贴，答案当然是可以的，只需要在粘贴时使用快捷键 Shift+option+Command+V，即可仅粘贴文字内容，不粘贴文字样式了。这项操作在 macOs 的系统应用，以及微软的 office 中都是有效的。

- **通过滚轮滚动查看应用的所有窗口​**
有时我们会用同一个应用打开好几个不同的窗口，例如同时打开几个Word文档，或者打开几个PPT幻灯片，此时如果想要查看这一个应用的全部窗口，比较简单的方法是在触控板上用四指向上轻扫，但问题是有不少小伙伴和我一样是鼠标党，把手从鼠标上拿开去触控板上划这么一下总感觉有一点麻烦，有没有什么更简单的办法呢？当然有，请出我们的老朋友：终端，在终端输入这行代码：​
​
defaults write com.apple.dock "scroll-to-open" -bool "true" && killall Dock​
​
按下回车，当你看到桌面图标和壁纸消失一下又出现说明命令生效了。此时，将鼠标光标放在程序坞的应用图标上，向下滚动一下滚轮，或者在触控板上用两个手指向上轻扫一下，这个应用的所有窗口就会平铺展示，再反方向滚动或轻扫一下他们就会消失，最方便的是它不需要你先切换到某个应用，光标指向谁就显示谁的全部窗口，整个操作过程丝滑连贯，效率大幅提升。​
